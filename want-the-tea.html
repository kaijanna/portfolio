Want the tea

Live site GitHub

Project description

Want The Tea is a social media front-end concept where users can browse posts, create new content, and view user profiles in a clean and engaging interface.

The project was built with HTML and Tailwind CSS, focusing on responsive layouts, reusable components, accessibility, and consistent visual design.

What I learned

This project helped me strengthen my understanding of Tailwind CSS, responsive design, layout systems, and creating consistent user interfaces across multiple pages.

Improvements Made

I revisited this project and improved the visual design, spacing and user experience. The updates included a softer color palette, more consistent card styling, improved navigation, enhanced post previews and a collapsible create-post section.
